# Cordic
The code can be run as follows:

Go into the source directories of the respective programs. 
Example: Vector 
cd TOPDIR/Src/Vector/src 
Compile the main.c file using gcc -o main main.c
Then run the program using ./main 
You will get 3 options:
- 1 to run without the non optimized implementation 
- 2 to run the fixed point arithmetic solution 
-3 to run the fully optimized solution
